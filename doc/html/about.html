About

ctools are a set of ftools-like executables needed for the scientific analysis of Cherenkov Telescope Array data. ctools are based on GammaLib, a versatile toolbox for the high-level analysis of astronomical gamma-ray data.


ctools are implemented in C++ and are fully scriptable using the Python scripting language.


The ctools development is led by the Insitut de Recherche en Astrophysique et Planétologie (IRAP), a mixed CNRS / Université Paul Sabatier laboratory in Toulouse, France.
