ctools reference manual¶

General information on FTOOLS usage can be found here.

Below you can find links to the command line tools help pages that are available in ctools.

ctools¶

  • ctbin — Calculates counts maps for binned maximum likelihood analysis.
  • ctlike — Performs binned and unbinned maximum likelihood analysis of CTA data.
  • ctmodel — Calculates model counts maps.
  • ctobssim — Observation simulator.
  • ctselect — Performs sub-selections of event data which are needed for unbinned maximum likelihood analysis.
  • ctskymap — Computes sky map from CTA data.

cscripts¶

  • cspull — Generates pull distribution for all model parameters.
  • cssens — Computes the CTA sensitivity for a number of energy bins.
  • cstsdist — Generates the TS distribution for a particular model.
